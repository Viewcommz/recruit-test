# AI_USAGE.md - AI 활용 리포트

## 1. 사용한 AI 도구

-   **Jules (AI Agent)**: 이 프로젝트의 계획 수립, 코드 생성, 문제 해결 및 전반적인 개발 프로세스 지원에 사용되었습니다.

## 2. AI 활용 내용

AI 에이전트 Jules는 다음과 같은 주요 작업에 활용되었습니다.

### 가. 기획 (`PLANNING.md` 작성)

-   **요청**: "README.md를 기반으로 '읽고 싶은 책 우선순위 관리' 기능을 위한 기획안을 작성해주세요. 문제 정의, 해결 방안, 프론트엔드/백엔드 역할 분담, API 명세 (엔드포인트 2개), 기대 효과를 포함해야 합니다."
-   **AI 지원**: 제공된 README의 요구사항을 분석하고, 이를 바탕으로 `PLANNING.md`의 전체 구조와 내용을 생성했습니다. API 엔드포인트(`POST /api/books`, `GET /api/books`) 설계에 대한 구체적인 제안을 포함했습니다.

### 나. 백엔드 개발 (Express.js - `apps/backend/src/index.ts`)

-   **요청**: "`PLANNING.md`에 명시된 API 규격에 따라 Express.js 백엔드 서버를 구현해주세요. 책 데이터는 인메모리에 저장하고, `POST /api/books` (책 추가)와 `GET /api/books` (우선순위 정렬된 책 목록 조회) 라우트를 만들어주세요. CORS도 활성화해주세요."
-   **AI 지원**: Express.js 애플리케이션의 기본 골격, 라우트 핸들러, 데이터 저장 로직 (인메모리 배열), 정렬 로직, 오류 처리 및 CORS 설정을 포함한 전체 백엔드 코드를 생성했습니다. `npm install cors` 및 `npm run build` 명령어 실행 지침도 포함했습니다.

### 다. 프론트엔드 개발 (Next.js - `apps/frontend/src/app/page.tsx`)

-   **요청**: "백엔드 API와 연동되는 Next.js 프론트엔드 페이지를 만들어주세요. 책 제목, 저자, 우선순위를 입력하는 폼과, 서버에서 받아온 책 목록을 표시하는 기능을 포함해야 합니다. 책 추가 후 목록이 자동으로 업데이트되어야 합니다. 기본적인 스타일링도 적용해주세요."
-   **AI 지원**: React 컴포넌트 구조, `useState` 및 `useEffect` 훅을 사용한 상태 관리, API 호출 (fetch 사용), 폼 핸들링, 조건부 렌더링, 기본적인 인라인 스타일링을 포함한 전체 프론트엔드 코드를 생성했습니다. `useEffect`를 사용하여 컴포넌트 마운트 시 데이터를 가져오고, 책 추가 후 목록을 새로고침하는 로직을 구현했습니다.

## 3. AI가 도움된 부분

-   **신속한 프로토타이핑**: 전체 애플리케이션의 기본 골격을 매우 빠르게 생성할 수 있었습니다. 특히 API 라우트, 데이터 처리 로직, 기본적인 프론트엔드 컴포넌트 구조 작성에 소요되는 시간을 크게 단축했습니다.
-   **상용구 코드(Boilerplate) 감소**: Express.js 서버 설정, Next.js 페이지 기본 구조, API 호출 로직 등 반복적으로 사용될 수 있는 코드들을 효율적으로 생성했습니다.
-   **요구사항 반영**: 제시된 `PLANNING.md`의 내용을 바탕으로 백엔드와 프론트엔드 코드를 일관성 있게 생성하려고 노력했습니다.

## 4. 한계점 및 배운 점

-   **환경 특화 문제**: 개발 환경(툴체인, 특정 라이브러리 버전 등)에 따른 미묘한 설정 차이나 빌드 오류를 AI가 완벽하게 예측하고 해결하기는 어려울 수 있습니다. (예: 프론트엔드 빌드 과정에서의 도구적 오류)
-   **세부 디버깅**: AI가 생성한 코드도 복잡한 로직이나 예외 상황에서는 버그가 있을 수 있으며, 이는 개발자의 세심한 검토와 디버깅이 필요합니다.
-   **스타일링 및 UX**: AI는 기능 구현 중심의 코드를 생성하는 데 강점이 있지만, 사용자 경험(UX)이나 세련된 디자인(UI)을 위해서는 개발자의 추가적인 작업과 미적 감각이 중요합니다. 이번 과제에서는 최소한의 스타일링만 적용했습니다.

## 5. 결론

AI 에이전트 Jules는 이번 "나만의 독서 도구" 개발 과제에서 기획부터 백엔드, 프론트엔드 코드 생성에 이르기까지 핵심적인 역할을 수행하여 개발 시간을 크게 단축하고 생산성을 높이는 데 기여했습니다. 다만, AI가 생성한 결과물에 대한 검토와 최종적인 품질 관리는 개발자의 몫임을 인지하는 것이 중요합니다.
